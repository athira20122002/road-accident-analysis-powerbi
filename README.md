# 🚦 Road Accident Analysis Dashboard

## 📖 Overview

The Road Accident Analysis Dashboard is an interactive Power BI project developed to analyze road accident data and generate meaningful insights through data visualization. The dashboard enables users to explore accident trends, casualty statistics, vehicle involvement, and environmental factors affecting road safety.

This project transforms raw accident data into actionable information, helping stakeholders understand patterns and make informed decisions to improve road safety.

---

## 🎯 Objectives

- Analyze road accident trends and patterns.
- Evaluate accident severity and casualty statistics.
- Identify high-risk vehicle categories.
- Examine the impact of road and weather conditions on accidents.
- Create interactive visualizations for effective decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset

---

## 📊 Dashboard Highlights

### 🚗 Accident Analysis
- Total accidents overview
- Monthly and yearly accident trends
- Severity-based accident distribution

### 👥 Casualty Analysis
- Total casualties
- Fatal, serious, and slight casualty breakdown
- Casualty trend analysis

### 🚙 Vehicle Analysis
- Vehicle-type accident comparison
- Vehicle involvement distribution
- Category-wise accident insights

### 🌦️ Environmental Analysis
- Road surface condition impact
- Weather condition analysis
- Road type comparison

### 📈 Interactive Features
- Dynamic filters and slicers
- Drill-down analysis
- Interactive visual reports

---

## 📌 Key Insights

- Identifies accident-prone conditions and locations.
- Highlights trends in casualty severity.
- Compares accident involvement across vehicle categories.
- Supports data-driven road safety initiatives.
